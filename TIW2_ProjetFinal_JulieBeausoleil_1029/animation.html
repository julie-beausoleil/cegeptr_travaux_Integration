ANIMATION D'IMAGES

Comme lorsque vous étiez plus jeunes et qu'on vous a fait dessiner des bonhommes alumettes sur plusieurs pages pour les tourner rapidement, le principe est de passer à travers d'une succession d'images rapidement pour donner l'impression de mouvement.

Previous Next

Préparer l'animation.

  • Une fois que tous vos calques sont prêts, ouvrez la fenêtre "Montage" dans Fenêtre > Montage.
  • Ajouter des images en appuyant sur le pour ajouter des images dans l'animation.
  • Activez ou désactiver le ou les calques au besoin.
  • Tester votre animation.
  • Exporter pour le web en format .gif. Fichier > Exportation > Enregistrer pour le web hérité avec les bons paramètres.
