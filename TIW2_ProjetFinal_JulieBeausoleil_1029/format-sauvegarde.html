FORMATS DE SAUVEGARDE

Chaque format de sauvegarde comporte ses avantages et ses inconvénients. Cliquer sur chaque items les découvrir !

AVANTAGES
  • Utilise la compression, fichier moins lourd.
  • Choix de mode colorimétrique de l'image
    • RVB, CMJN.
INCONVÉNIENTS
  • Utilise la compression avec perte, attention de ne pas trop compresser !
  • Ne permet pas la transparence.

Idéal pour les images vectorielles

  • Les images vectorielles peuvent être redimensionnées sans perte de qualité, tandis que ce n'est pas possible avec des images matricielles.
  • Il peut être sauvegardé en tant que fichier éditable dans un logiciel d’image vectorielle (Adobe Illustrator) ou peut être exporté en tant que code (langage de balisage XML), qui sera interprété par votre navigateur.

Idéal pour la vidéo

  • Le MPEG-4, un standard destiné à permettre le codage de données multimédia sous formes d'objets numériques, afin d'obtenir une plus grande interactivité, ce qui rend son usage particulièrement adapté au Web et aux périphériques mobiles.
  • C’est un format qui permet la sauvegarde de vidéo et d’audio avec compression.
En savoir plus >

L'EXPORTATION

  • Pour les formats de sauvegarde matriciels pour le web, à partir de Photoshop Fichier > Exportation > Enregistrer pour le web (hérité) avec les bons paramètres.
  • Pour le format de sauvegarde vectoriel pour le web, à partir d'Illustrator Fichier > Exportation la sélection avec les bons paramètres.
  • Pour le format de sauvegarde vidéo en .mp4, à partir de Photoshop, dans le bas de la fenêtre Montage ou Fichier > Exportation > Rendu vidéo avec les bons paramètres.
